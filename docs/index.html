Description

Portfolio websites serve as an important medium for artists to present their works. But beyond its aesthetic purpose, building it from scratch can also be a good resource to understand the core concepts of modern javascript frameworks.

In this 1 point 3-week course, we will explore the full lifecycle of static portfolio site development from initial setup through final deployment using Vue.js. Students are expected to design responsive UIs and layouts, build reusable components, and utilize advanced methods.

This course is programming-heavy and suitable for students seeking to create a custom website or learn the basics of Vue.js (with Nuxt.js). Students must take ICM and at least one other javascript programming course. The course assumes that students have a solid understanding of HTML, CSS, JavaScript, npm, Git.

Course Objectives

At the completion of this course, the students will:

  1. Be comfortable designing visuals/UX for static web application
  2. Learn modular programming patterns in javascript using Vue.js
  3. Build and deploy a custom portfolio website (or equivalent)

Schedule

  • Intro
    • Some creative portfolio websites

    • javascript frameworks
  • Vue basics
    • Vue explained in 100s
    • >Nuxt code structure

    • Lifecycle hooks (created, mounted, unmounted)

    • Navigation with NuxtLink
  • Directives
    • v-bind (:) - dynamic data (src, class, id, key)
    • v-if - render elements conditionally
    • v-for - loop through data

Week 2 - Responsive Layout / Styles / Libraries

  • Responsive Layout
    • Scss basics and setup in Nuxt

  • Styles
    • Configure global style
    • Transition between pages
    • Bootstrap UIs
  • Libraries
    • Create global function
    • Use p5.js/three.js in component

Week 3 - Events / Deploy / Presentations

  • Events
    • Emit events to parent
    • Event bus in Nuxt
  • Deploy
    • Static site generation
    • Gitpage
    • Netlify
    • SFTP (dreamhost)
  • Presentation

Evaluation

You are required to attend all class meetings and submit all weekly assignments and a final project.

Grading (pass/fail) will be based on a combination of factors:

  • Attendance, participation in class discussion, and engagement in other students' projects (40%)
  • Assignments (40%)
  • Final Project (20%)

Please see ITP's statement on Pass/Fail which states that a "Pass" is equivalent to an "A" or a "B" while anything less would be considered a "Fail".

Attendance is mandatory. Please inform your teacher via email if you are going to miss a class. Two unexcused absences is cause for failing the class. (An unexcused lateness of 10 minutes or more is equivalent to 1/2 an absence.)

This class will be participatory, you are expected to participate in discussions and give feedback to other students both in class and participate with their projects. This (along with attendance) is 40% of your grade.

Class will culminate with final projects. You are expected to push your abilities to produce something that utilizes what you have learned in the class that is useful in some manner to yourself or the world. This will comprise 20% of your grade.
